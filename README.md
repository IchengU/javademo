本项目是我学习 Java 的练习代码仓库。主要记录从基础语法到面向对象编程的学习过程，包含每天的练习和项目。

## 🎯 项目目的

- 系统学习 Java 核心语法和面向对象编程
- 练习使用 Git 进行版本控制
- 积累代码量，为后续框架学习打下基础

## 🚀 如何运行

项目使用 JDK 17，可以直接在 IntelliJ IDEA 中打开运行。每个练习有单独的类，直接运行对应的 `main` 方法即可。
